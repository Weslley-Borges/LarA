## Dependencies
- Install [Node.js](https://nodejs.org/) (latest version)
- Install [Python]() (version 3.9.6)
  > NOTE: You only need to install Python if you want to use Lara's API.
- Install all dependencies:

   ```npm install```

   ```pip install -r requirements.txt```
